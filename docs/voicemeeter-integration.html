Phase 3 (будущее) — audio pipeline с noise cancellation. ~35-45ms задержка, но всё ещё ниже порога восприятия для речи.

Сравнение: Zoom/Discord добавляют ~50-150ms собственной задержки. Наш пайплайн незаметен на их фоне.

Дорожная карта

Phase 1 — Voicemeeter Remote Control
Пады: mute, EQ, routing, gate, comp. Крутилки: gain, gate, comp. Визуализатор маршрутизации с level meters. Подключение через VoicemeeterRemote64.dll.
Phase 2 — Soundboard через Voicemeeter
Направить Sample Player на Voicemeeter AUX Input. Кнопка Send2Mic отправляет звуки собеседнику. Визуализация: видно, когда звук идёт в B1.
Phase 3 — Noise Cancellation
Real-time audio pipeline: Yeti → Python (RNNoise/noisereduce) → VB-CABLE → Voicemeeter. Крутилкой регулируешь интенсивность. Toggle по кнопке.
Phase 4 — Voice Changer (AI)
Подключение AI-модели для трансформации голоса в реальном времени. Пресеты на падах (Deep, Chipmunk, Robot). Pitch shift через крутилку.
