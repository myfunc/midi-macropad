# Настройка сцен OBS

Пошаговое руководство по созданию 4 сцен в OBS Studio, а также настройка Replay Buffer и аудио.

---

## Сцена 1: Screen+PiP (Шеринг экрана + вебка)

Захват окна приложения с наложением веб-камеры в углу.

### Шаги

1. Создать новую сцену, назвать **"Screen+PiP"**
2. Добавить источник **"Window Capture"**:
   - Выбрать нужное приложение из списка
   - Правый клик -> **Transform** -> **Edit Transform**
   - Установить **Crop Left = 400**, **Crop Right = 400** (обрезает по 400 пикселей с каждой стороны)
3. Добавить источник **"Video Capture Device"** (веб-камера):
   - Уменьшить размер примерно до **25%** от оригинала
   - Расположить в **правом нижнем углу**
   - При необходимости обрезать края камеры: зажать **Alt** и перетащить край рамки
4. Убедиться, что веб-камера находится **выше** захвата экрана в списке источников (наложение поверх)

## Сцена 2: Half Screen (Половина экрана)

Для ультраширокоформатных мониторов -- показываем только половину экрана.

### Шаги

1. Создать новую сцену, назвать **"Half Screen"**
2. Добавить источник **"Display Capture"**
3. Правый клик на источнике -> **Transform** -> **Edit Transform**
4. Настроить обрезку:

**Для ЛЕВОЙ половины экрана:**

| Параметр | Значение |
|---|---|
| Crop Right | Половина ширины монитора |

**Для ПРАВОЙ половины экрана:**

| Параметр | Значение |
|---|---|
| Crop Left | Половина ширины монитора |

### Значения обрезки для популярных ультрашироких разрешений

| Разрешение монитора | Обрезка (Crop) |
|---|---|
| 3440x1440 | 1720 |
| 2560x1080 | 1280 |
| 5120x1440 | 2560 |

## Сцена 3: Camera (Вебка)

Полноэкранный вид с веб-камеры.

### Шаги

1. Создать новую сцену, назвать **"Camera"**
2. Добавить источник **"Video Capture Device"**
3. Выбрать устройство веб-камеры
4. **Resolution/FPS Type** = **Device Default**
5. Растянуть на весь холст: правый клик -> **Transform** -> **Fit to Screen**

---

## Сцена 4: Cam+App (Вебка 80% + Приложение 20%)

Комбинированная сцена: камера занимает 80% высоты, окно приложения -- 20%. По бокам декоративные полосы.

Расчёт для холста **1920x1080**:

```
+--------+-----------------------------+--------+
|        |                             |        |
|  192px |    Веб-камера (1536x864)    | 192px  |
| левый  |         80% высоты          | правый |
| бордюр |                             | бордюр |
|        |                             |        |
+--------+-----------------------------+--------+
|        |  Приложение (1536x216)      |        |
|        |         20% высоты          |        |
+--------+-----------------------------+--------+
```

### Шаги

1. Создать новую сцену, назвать **"Cam+App"**

2. Добавить **"Video Capture Device"** (веб-камера):

   | Параметр | Значение |
   |---|---|
   | Position X | 192 |
   | Position Y | 0 |
   | Size | 1536 x 864 |

3. Добавить **"Window Capture"** (окно приложения, например, музыкальный плеер):

   | Параметр | Значение |
   |---|---|
   | Position X | 192 |
   | Position Y | 864 |
   | Size | 1536 x 216 |

4. Добавить два источника **"Color Source"** для боковых полос:

   **Левый бордюр:**

   | Параметр | Значение |
   |---|---|
   | Position X | 0 |
   | Position Y | 0 |
   | Size | 192 x 1080 |

   **Правый бордюр:**

   | Параметр | Значение |
   |---|---|
   | Position X | 1728 |
   | Position Y | 0 |
   | Size | 192 x 1080 |

   Цвет: выбрать тёмный градиент или тематический цвет.

5. 10% от 1920 = 192px на каждую боковую полосу.

## Настройка Replay Buffer

Позволяет сохранять последние N секунд записи по нажатию кнопки.

### Шаги

1. Открыть **Settings** -> **Output** -> **Replay Buffer**
2. Поставить галочку **Enable Replay Buffer**
3. Установить **Maximum Replay Time** = **600 секунд** (10 минут)
4. Сохранение реплеев запускается из **Midi Macropad** -- горячая клавиша в самом OBS не нужна
5. Файлы сохраняются в `~/Videos/OBS-Replays/`

> **Примечание:** убедитесь, что папка `OBS-Replays` существует, или OBS создаст её автоматически при первом сохранении.

## Стерео и аудио

| Настройка | Где | Значение |
|---|---|---|
| Каналы | Settings -> Audio -> Channels | **Stereo** |
| Дорожка записи | Settings -> Output -> Recording -> Audio Track | 1 (или несколько для многодорожечной записи) |
| Формат записи | Settings -> Output -> Recording -> Recording Format | **MKV** |

### Почему MKV?

- Устойчив к вылетам -- файл останется читаемым, даже если OBS упадёт
- Поддерживает несколько аудиодорожек
- Легко конвертируется в MP4 через OBS: **File** -> **Remux Recordings**
